Create A Service Project
	http://msdn.microsoft.com/en-us/library/zt39148a(v=vs.110).aspx
	File > New Project > Visual # > Windows > Windows Service

How To Run A Service
	1)	Build project

	2)	Install the service
			Run 'Developer Comand Prompt for VS13' (installed with VS13)
			$ cd '..\<proj-name>\<proj-name>\bin\Debug'
			$ installutil <proj-name>.exe

	3)	Start/Run/Stop the service
			Run 'services.msc'
			Find <service-name>, start the service
			Stop <service-name>

	4)  Uninstall the service
			Run 'Developer Comand Prompt for VS13'
			$ cd '..\<proj-name>\<proj-name>\bin\Debug'
			$ installutil /u <proj-name>.exe
